למה זה מעניין?

בכל פעם שאתם משתמשים ב-ChatGPT, השיחה שלכם עוברת דרך השרתים של OpenAI. אותו דבר עם Claude, Gemini ושאר השמות. זה נוח ועובד מצוין, אבל יש דברים שלא נעים לשלוח לחברה אמריקאית: חוזים, תיקים רפואיים, מכתב מעורך הדין, רעיונות עסקיים שעדיין לא רוצים שיצאו החוצה.

הפתרון: להריץ AI על המחשב שלכם. בלי אינטרנט. בלי שאף אחד יראה. חינם.

שני הכלים שנתקין

🦙 Ollama, המנוע

תוכנה שמורידה ומריצה מודלי AI (כמו Llama של מטא, Qwen של עליבאבא, Mistral מצרפת) על המחשב שלכם. בלי ממשק יפה, רק מנוע.

🌐 Open WebUI, הממשק

תוכנה שמתחברת ל-Ollama ונותנת לכם ממשק שנראה בדיוק כמו ChatGPT, עם היסטוריית שיחות, העלאת קבצים, חיפוש בהן, תיקיות, וכל הטוב.

הרעיון: Ollama מריץ את ה-AI ברקע. Open WebUI נותן לכם דף אינטרנט יפה לדבר איתו.

1התקנת Ollama

נכנסים לאתר הרשמי ומורידים את ההתקנה למערכת שלכם:

👉 ollama.com/download

Windows / Mac

מורידים את הקובץ, לוחצים פעמיים, מאשרים. זהו. אחרי ההתקנה, Ollama רץ ברקע אוטומטית.

Linux

curl -fsSL https://ollama.com/install.sh | sh

בדיקה שהכל עובד

פותחים Terminal (במק) או PowerShell (בווינדוס) ומקלידים:

ollama --version

אם רואים מספר גרסה, Ollama מותקן ועובד 👍

2הורדת מודל ראשון

עכשיו צריך להוריד מודל AI. יש המון לבחור, הנה כמה מומלצים למתחילים:

llama3.2 מטא, איזון טוב למתחילים 2GB
qwen2.5 עליבאבא, תומך עברית מצוין 4.7GB
mistral צרפתי, מהיר וקליל 4.1GB
gemma4 גוגל, החדש מאפריל 2026, חזק במיוחד 3.3GB+

להורדת המודל, מקלידים ב-Terminal:

ollama pull llama3.2

ההורדה לוקחת כמה דקות (תלוי באינטרנט). אחרי שזה גמור, אפשר לדבר עם המודל מהטרמינל:

ollama run llama3.2
טיפ: לעברית, qwen2.5 עובד הכי טוב מבין הקטנים. llama3.2 מבין עברית אבל עונה הרבה פעמים באנגלית.

3התקנת Open WebUI

עכשיו הכיף, נתקין את הממשק שיחליף את הטרמינל המכוער.

הדרך הכי קלה: אפליקציית Desktop

החל מגרסה 0.9.0 (אפריל 2026), Open WebUI יצא כאפליקציית מחשב רגילה, בלי שום הגדרה מסובכת:

👉 openwebui.com, מורידים, לוחצים פעמיים, גמרנו.

הדרך הקלאסית: דרך Python

אם אתם רוצים את הגרסה שרצה בדפדפן (יותר תכונות מתקדמות):

pip install open-webui
open-webui serve

אחרי שזה רץ, פותחים את הדפדפן ונכנסים ל-http://localhost:8080

נדרש Python 3.11+ מותקן על המחשב. בווינדוס, כדאי להתקין מ-python.org ולסמן "Add to PATH" בהתקנה.

4הצ'אט הראשון

  1. פותחים את Open WebUI (אפליקציה או דפדפן)
  2. נרשמים, שם משתמש וסיסמה. זה נשמר רק על המחשב שלכם.
  3. למעלה במרכז יש בורר מודלים, בוחרים את המודל שהורדתם (llama3.2 או מה שיש לכם)
  4. מתחילים לשאול שאלות 🎉
נסו: לחצו על האייקון של מהדק הנייר ⌘ העלו PDF, חוזה, או מסמך, ותשאלו עליו שאלות. זה נקרא RAG וזה אחת התכונות הכי שימושיות.

מה עוד אפשר לעשות?

📚 בסיסי ידע

מעלים תיקייה שלמה של מסמכים, והמודל יודע לחפש בהם. אידיאלי לעו"ד עם תיקי לקוחות, או רואה חשבון עם ניירת.

🤖 יצירת בוטים מותאמים

בונים בוט עם אישיות מסוימת, גישה לקבצים מסוימים, וחוקים מסוימים. למשל בוט שעוזר לילדים שלכם בשיעורי בית.

🎙️ דיבור

מדברים למיקרופון, המודל עונה בקול. עובד גם בעברית.

🌐 חיפוש באינטרנט

אפשר לחבר חיפוש מהאינטרנט (כמו פרפלקסיטי), עדיין מקומי.

🎨 יצירת תמונות

עם תוסף נוסף (ComfyUI), אפשר לייצר תמונות מקומית, בלי DALL-E ובלי מידג'רני.

שורה תחתונה

למה כן?

  • פרטיות מוחלטת, שום דבר לא יוצא מהמחשב
  • חינם, ללא מנוי, ללא הגבלות
  • עובד גם אופליין
  • אפשר להעלות מסמכים רגישים בלי דאגה

למה לא?

  • המודלים החינמיים לא ברמה של GPT-5 או Claude, הם טובים, אבל פחות חכמים
  • דורש מחשב סביר (16GB RAM לפחות)
  • ההתקנה דורשת קצת סבלנות בפעם הראשונה
  • דורש מקום בדיסק (כל מודל = כמה GB)
ההמלצה שלי: זה לא במקום ChatGPT, זה בנוסף. השתמשו ב-AI הענן לדברים מורכבים, וב-AI המקומי לכל מה שרגיש.
